# CMPD_RacialBiasAnalysis

Analyzed CMPD Traffic stop data to identify trends and racial disparities in traffic stops and searches within the city of Charlotte and trained predictive models using hyperparameters in H20 R package.


The original file is CMPD_Traffic_Stops.csv. 
Performed one hot encoding on various variables and created another csv file with those derived variables. That file is CMPD_binary.csv
